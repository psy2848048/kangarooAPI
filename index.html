POST /api/v1/login¶
로그인
Parameters
  1. “email”: String, email
  2. “password”: String, sha256(password) 값을 전송
  3. “machine_type”: (OPTIONAL) 기기 타입 (android, ios), 푸시 알람때문에 만듬
  4. “machine_id”: (OPTIONAL) 기기 ID
Response
  1. 200: OK
  2. 403: Fail
